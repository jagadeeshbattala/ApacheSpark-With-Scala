## Scala Setup using Coursier

Follow this page to setup scala - https://get-coursier.io/docs/cli-installation

Run below commands to install Scala in Mac
$ brew install coursier/formulas/coursier
$ cs setup

After install this, you are ready with scala setup, but to use scala REPL, you need Java 17.
If you have Java 11 setup, you can launch the scala REPL using -> cs launch scala:2.13.12

There you go -> You can start using REPL.
